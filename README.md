# 🐶 하루멍록이란?

**하루멍록**은 반려동물의 일상과 건강을 체계적으로 기록하고 관리할 수 있는 반려인 전용 플랫폼입니다.  
식사, 투약, 산책 같은 반복적인 루틴을 기록하고, 커뮤니티를 통해 반려인들 간의 소통도 가능하며,  
**알림과 추천 기능을 통해 더 건강하고 즐거운 반려생활을 도와줍니다.**

> "오늘 하루도, 반려동물과 함께한 모든 순간을 기록해보세요!"

### ✅ 주요 기능
- 🐾 **반려동물 프로필 등록 및 관리**
- 📅 **캘린더 기반의 일정 및 루틴 관리**
- 🍚 **식사 / 💊 투약 / 🚶 산책 기록 및 알림 기능**
- 🗺️ **산책 경로 추천 및 이동 경로, 사진 기록**
- 💬 **반려인 커뮤니티 기능 (게시글, 댓글, 좋아요 등)**

# 📋 Commit Message Convention (이모지 + 소문자 적용)
| Tag           | Description             |
|---------------|-------------------------|
| `✨ feat`      | 새로운 기능 추가        |
| `🐛 fix`      | 버그 수정              |
| `📝 docs`     | 문서 추가, 수정, 삭제  |
| `🧪 test`     | 테스트 코드 추가, 수정, 삭제 |
| `🎨 style`    | 코드 형식 변경         |
| `♻️ refactor` | 코드 리팩토링         |
| `⚡ perf`      | 성능 개선              |
| `🏗️ build`   | 빌드 관련 변경사항     |
| `👷 ci`       | CI 관련 설정 수정     |
| `🚀 chore`    | 기타 변경사항          |
